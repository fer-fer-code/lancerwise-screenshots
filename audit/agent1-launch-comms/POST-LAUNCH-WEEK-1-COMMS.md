## Engagement strategy by channel

### Twitter / X

| Pattern | When | Goal |
|---|---|---|
| Quote-RT thoughtful replies | Anytime | Amplify users without their permission threshold |
| Reply to every signed-up user mention | First 7 days | Build founder approachability signal |
| Post screenshots of bug fixes shipped within hours | Day 1-5 | Show responsiveness |
| Daily "what I shipped today" thread | First 7 days, then 2-3×/week | Build-in-public flywheel |
| Avoid retweeting praise without comment | Always | Avoid looking promotional |

### LinkedIn

| Pattern | When | Goal |
|---|---|---|
| Founder-voice posts с specific learnings | 2-3× в week 1 | Builds founder/product affinity |
| Comments on existing freelance / SaaS / solopreneur conversations | Daily 5-10 min | Discovery via reciprocity |
| Avoid spam-style comment-and-link | Always | LinkedIn algorithm down-ranks |

### Reddit

| Pattern | When | Goal |
|---|---|---|
| Answer questions в r/freelance с no link (just helpful) | Daily 5 min | Build karma + credibility |
| Post launch announcement | T+30min only | Don't repeat-post |
| If asked specifically, link к LancerWise in а comment with disclosure | Reactive only | Stays within "be helpful" mod policy |
| Avoid posting same content across multiple subs | Always | Reddit anti-spam |

### Hacker News

| Pattern | When | Goal |
|---|---|---|
| Show HN — text post (no link-only) | T+2h, US PT 6-9am | Peak hours optimization |
| Reply к every top-level comment thoughtfully | First 24h | HN values founder engagement |
| Don't ask family/friends к upvote | Always | HN flags + shadow-bans |
| If asked technical questions, give specific answers | Always | Tech credibility on HN |

### Product Hunt

| Pattern | When | Goal |
|---|---|---|
| Soft launch Tuesday timing | T+24h to T+48h | Highest discovery day |
| First comment from maker (template в ANNOUNCEMENT-DRAFT.md) | Within first hour | Sets thread tone |
| Reply к every comment с specific answers | First 24h | Maker engagement scoring |
| Avoid "we" language solo founders | Always | Authenticity |

## What к NOT do в week 1

| Don't | Why |
|---|---|
| Pay для bot accounts / fake reviews | Anti-policy on all platforms; reputational risk |
| Make claims you can't back up ("most popular freelance CRM") | Per `feedback_marketing_honesty_policy.md` |
| Run paid ads на day 1 | Wait until organic baseline established (week 2+) |
| Hide bugs publicly while fixing privately | Builds-in-public credibility = transparency |
| Promise features that don't exist ("mobile app coming next month") | Without commitment, don't tease |
| Engage с trolls | Block / mute / move on |
| Reply к support questions in public threads (если sensitive) | Move к DM / email |
