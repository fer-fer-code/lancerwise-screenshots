## 4. Шаблоны ответов

**Ответ 1 — "Чем отличается от [конкурент]?"**
```
Три честных различия:
1. Free тариф навсегда — у большинства базовый функционал за $15–25/мес
2. ИИ встроен в основные сценарии (договоры/брифы/письма/советник), а не отдельный чат-бот
3. Кодовая база 2026 года, без legacy — быстрее и легче

Попробуй и сам реши. Карта не нужна.
```

**Ответ 2 — "Только что зарегался, выглядит круто!"**
```
Спасибо 🙏 Если в первые 24 часа поймаешь что-то странное — ответь сюда или в личку. Сегодня инбокс мой, починю до завтра.
```

**Ответ 3 — "Какой стек?"**
```
Next.js 14 + Supabase (Postgres с Row-Level Security) + Vercel. ИИ — multi-provider (Gemini Flash, Claude Haiku, Groq), роутится по задаче. Платежи через LemonSqueezy. PWA фронт — ставится как приложение.
```

**Ответ 4 — "Безопасно ли хранить данные?"**
```
RLS на уровне БД — даже если в коде приложения баг, БД блокирует кросс-тенантные чтения. Полный GDPR-экспорт из настроек: данные уходят с тобой, если решишь уйти. Не продаём, не делимся.
```

**Ответ 5 — "Перехожу с [Bonsai/другое] — можно мигрировать?"**
```
Bulk CSV-импорт клиентов работает уже сейчас. Импорт инвойсов в дорожной карте. Если застрял на полпути миграции — пиши в личку, помогу руками. Хочу, чтобы это было легко.
```
