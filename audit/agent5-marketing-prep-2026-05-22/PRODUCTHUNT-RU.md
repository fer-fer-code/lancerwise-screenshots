## 2. Краткое описание (~260 знаков)

LancerWise заменяет 7 сервисов, между которыми скачет каждый фрилансер — инвойсы, договоры, CRM, тайм-трекер, предложения — одним рабочим пространством. ИИ пишет договоры, делает брифы из заметок, составляет письма клиентам. Бесплатно навсегда для соло-работы.

## 4. Какие проблемы решает

- **Зоопарк инструментов** — заменяет 5–7 разных сервисов на один воркспейс
- **Невидимая нагрузка от админки** — автоматизирует рутину: рекуррентные инвойсы, напоминания об оплате, генерация договоров, еженедельные инсайты
- **ИИ, который реально полезен** — не чат-бот ради галочки, а ИИ внутри ключевых сценариев: договоры, брифы, письма клиентам, оценка риска просрочки
- **Скрытые потери денег** — показывает несписанные часы, просроченные инвойсы, клиентов, до которых стоит дотянуться, проекты, выбежавшие за бюджет
- **Нулевой барьер входа** — бесплатный тариф навсегда, можно не вкладываться в инструмент, пока нет стабильного дохода

## 5. Как это поможет пользователю

- **Экономия 5–10 часов рутины в неделю** — переключение между инструментами съедает именно столько у среднего фрилансера
- **Деньги приходят быстрее** — автонапоминания, частичные оплаты, онлайн-ссылки на оплату, алёрты по просрочкам
- **Клиенты подписывают быстрее** — договоры готовы за минуту, флоу принятия предложения работает с телефона
- **Лучше деловые решения** — еженедельные ИИ-инсайты на реальных данных. Советник отвечает на "стоит ли поднять ставку?" с цифрами
- **Работа откуда угодно** — PWA, мобилка, 16 валют, RU + EN интерфейс

## 6. Топ-10 ожидаемых вопросов

(Сокращённо — полный список см. EN-версию, переведено по смыслу)

**Чем отличается от Bonsai/Dubsado/HoneyBook?**
Бесплатный тариф навсегда, ИИ встроен в основные сценарии (не маркетинговый ярлык), легче и быстрее интерфейс — кодовая база 2026 года, без legacy.

**Сколько стоит?**
Free ($0, 2 клиента), Pro ($15/мес или $12 при годовой оплате, 14 дней триал), Business — coming soon.

**Где хранятся данные?**
Supabase Postgres с Row-Level Security на каждой таблице. Хостинг — Vercel. Полный GDPR-экспорт из настроек.

**Какой ИИ под капотом?**
Многопровайдерный роутинг: Gemini Flash для быстрых задач (брифы, письма), Claude Haiku для договоров и рассуждений, Groq для чата с низкой задержкой.

**Есть мобилка?**
PWA, ставится из браузера. Нативные iOS/Android — в дорожной карте.

**Только для дизайнеров/фотографов?**
Нет, схема общая. Работает для разработчиков, копирайтеров, консультантов, коучей.

**Импорт данных из другого инструмента?**
Bulk CSV для клиентов есть. Инвойсы — в дорожной карте. Мигрируешь с Bonsai/HoneyBook — напиши, помогу руками.
